# NIST SP 800-22 input files

This directory contains binary stream exports for external randomness testing.

NIST SP 800-22 execution depends on the locally compiled STS suite and its expected input path conventions. Record the STS version, compiler, operating system, bitstream length, and all selected tests before importing results.

Do not treat SP 800-22 results as cryptographic proof. These outputs are conventional statistical diagnostics only.
