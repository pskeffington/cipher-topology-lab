\section{Discussion}

The interpretation will focus on whether topological features separate weak or structured controls from modern-cipher and random-baseline conditions. A useful outcome would show that persistent-homology features are sensitive to known weak controls while avoiding unsupported claims about the security of standardized ciphers. A null result would still be informative if the replication package clearly demonstrates that the tested topological features do not add discriminative value beyond conventional diagnostics under the studied conditions.
