\section{Discussion}

The 64-replicate internal run provides evidence that the topological pipeline can recover at least one designed weak-generator positive control. The clearest result was the separation of the LCG condition from the SHA-256 seeded baseline under byte-pair H0 summaries. This pattern appeared in both persistence entropy and Euclidean distance-to-baseline, indicating that the connected-component structure induced by the byte-pair point cloud was sensitive to the LCG stream structure.

The same run did not show comparable separation for the AES-CTR deterministic-output conditions, OS CSPRNG output, or xorshift32 under the tested embeddings and feature set. This negative result is important for the interpretation of the method. The current evidence supports the pipeline as a diagnostic that can detect a specific weak positive control, but it does not support claims that these summaries distinguish AES-CTR output from the deterministic SHA-256 baseline in the present design. The xorshift32 result also suggests that not every weak generator will be detected by the current representation and feature set.

The distinction between the SHA-256 reproducibility baseline and OS CSPRNG sensitivity condition remains central. Distance-to-baseline summaries should be read as distances from the configured deterministic baseline, not as proof that any stream is cryptographically random. OS CSPRNG comparisons are useful for sensitivity analysis but should not be treated as exactly reproducible reference values across independent executions.

Several limitations follow from the current pre-release design. The external randomness-test table remains unpopulated, so the manuscript presently supports only the internal topological and lightweight randomness-diagnostic analysis. The configured embeddings are intentionally narrow, limited to byte-pair and sliding-window representations. Future runs should add imported Dieharder or NIST SP 800-22 results, additional replicate scales, and explicitly configured cubical-complex experiments before broader claims are made about diagnostic coverage across cipher families or weak-generator classes.
