# Publication Plan

## Working title

Topological Diagnostics of Symmetric-Cipher Output Randomness

## Contribution type

This is a reproducible computational-diagnostics paper. It does not propose a new cipher, new attack, or proof of security.

## Defensible contribution

The defensible contribution is a transparent pipeline for comparing persistent-homology summaries against conventional randomness diagnostics under controlled cipher-output and weak-control conditions.

## Minimum publishable package

- Local stream generation with manifests and SHA-256 digests
- AES-128 baseline
- Ascon extension
- OS CSPRNG baseline
- Weak generator controls
- At least two embedding methods
- Persistent homology features for H0 and H1
- Internal diagnostics and external randomness-test comparison
- Replication instructions
- Cautious manuscript framing

## Venue fit

Likely targets include applied cryptography workshops, cybersecurity engineering venues, computational topology workshops, reproducibility tracks, and preprint release through IACR Cryptology ePrint or arXiv.
